## 单页应用中的页面组织核心思想

| 特点                     | 说明                                                  |
| ------------------------ | ----------------------------------------------------- |
| **一个 HTML 文件** | 入口通常是 `index.html`，只加载一次                 |
| **前端路由控制**   | 使用前端框架的路由库（如 React Router、Vue Router）   |
| **页面切换不刷新** | 切换页面只改变 URL 的 hash 或 history，不重新加载页面 |
| **按组件组织页面** | 每个“页面”是一个组件或组件集合                      |
| **支持嵌套路由**   | 页面中可以有子页面或区域切换                          |
