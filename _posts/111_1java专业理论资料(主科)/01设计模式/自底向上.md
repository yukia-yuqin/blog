
## `requests.post()`

| 模式                 | 在 `requests.post()`中的作用               |
| -------------------- | -------------------------------------------- |
| **外观模式**   | 封装了复杂的 HTTP 调用细节，用户使用简单接口 |
| **构建者模式** | 高级用法中可以分步骤构造请求                 |
| **工厂模式**   | 内部根据方法创建不同类型的请求对象           |
| **模板方法**   | 请求流程是可扩展的固定模板                   |
